## 🤔 Seedance 2.0 とは？

**Seedance 2.0** は**ByteDance**が開発した動画生成モデルで、業界初の**4モーダル同時入力**をサポートするモデルです — 画像、動画、音声、テキスト。

**Key Features:**
- 🎥 **テキストから動画** — テキスト説明から動画を生成
- 🖼️ **画像から動画** — 静止画をダイナミックな動画にアニメーション化
- 📹 **動画から動画** — 既存の動画を変換・拡張
- 🎵 **音声駆動** — 音声入力で動画を生成
- 📐 **最大1080p解像度**、4〜15秒の長さ
- 🔊 **自動吹き替え＆スコアリング** — 自動ナレーションとBGM

### Seedance 2.0 馬の新年動画プロンプト

![中文](https://img.shields.io/badge/lang-中文-red)

> Seedance 2.0 を使用して AI が生成した動画の詳細なプロンプト。中国の旧正月（午年）を祝う馬を描写しています。プロンプトでは、一連のアクション（ゆっくりから速く、クライマックス）、カメラの動き、コミック調のスタイル、花火や提灯などの背景要素、そして最後のお祝いのテキストが指定されています。

#### 📝 プロンプト

```
さまざまなポーズの参考画像をアップロードしてください：【@Image 1】【@Image 2】【@Image 3】【@Image 4】【@Image 5】。馬のイメージは一貫性を保ち、馬のイメージは一貫性を保ってください。馬の登場順序は調整可能です ✅
コアロジックのアクションシーケンス：スロー → ファスト → クライマックス、アクションは小さくから大きく → フリーズフレーム
Image 1：立つ/少し見上げる
Image 2：見上げる/首を回す
Image 3：速歩/ひづめを上げる
Image 4：加速して走る
Image 5：跳躍/ジャンプ
Image 5：頭を高く上げたフリーズフレーム（字幕表示）
アクションの詳細：手足、たてがみ、尾は自然でダイナミックに。コミック風に少し誇張してもOK。各ポーズ間のスムーズな移行、連続したアクション。
カメラ：前半はゆっくり、わずかな回転 + プッシュ/プル。後半はアクションが加速し、ダイナミズムを強調。一貫した方向性を保ち、無作為な動きは避ける。
背景と雰囲気：コミック風の花火、遠景。金色の光の点/小さな星が空中に浮かぶ。簡略化された赤い提灯の前景要素を追加可能。
音楽とリズム：陽気なドラムビートまたは軽快な正月音楽。切り替えのアクションタイミングを調整。
```
