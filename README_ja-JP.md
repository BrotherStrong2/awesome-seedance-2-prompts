## 🤔 Seedance 2.0 とは？

**Seedance 2.0** は**ByteDance**が開発した動画生成モデルで、業界初の**4モーダル同時入力**をサポートするモデルです — 画像、動画、音声、テキスト。

**Key Features:**
- 🎥 **テキストから動画** — テキスト説明から動画を生成
- 🖼️ **画像から動画** — 静止画をダイナミックな動画にアニメーション化
- 📹 **動画から動画** — 既存の動画を変換・拡張
- 🎵 **音声駆動** — 音声入力で動画を生成
- 📐 **最大1080p解像度**、4〜15秒の長さ
- 🔊 **自動吹き替え＆スコアリング** — 自動ナレーションとBGM

### 超リアルなイヌワシのシネマティックシーケンスプロンプト

![English](https://img.shields.io/badge/lang-English-blue)

> Seedance 2.0 用の、詳細かつ複数段落にわたるプロンプト。金色のワシが崖から飛び立ち、都市環境を飛び回り、屋上の看板に着地する、超リアルな映画のようなシーケンスを要求するもので、リアリズム、物理学、カメラワークを重視している。

#### 📝 プロンプト

```
夜明け、岩だらけの崖から飛び立つイヌワシの超リアルなシネマティック シーケンス。翼が広がり、羽毛が一本一本分かれ、空気抵抗を受けているのが見て取れる。鳥は前方に滑空し、吊り下げられた建設クレーンのケーブルを避けるため左に旋回した後、屋上のアンテナ群をかわすため急上昇する。空気抵抗を受けて翼がわずかにしなり、初列風切羽が独立して曲がり、リアルな気流との相互作用が表現されている。間近を通過する際、冷たい朝の空気の中にわずかに息の蒸気が見える。

カメラは安定した空中リグを使用し、中距離で並行して追跡する。風の乱気流によるわずかな自然な振動がある。背景には、朝靄が広がる高層ビル群のスカイラインが見え、低い東からの太陽光がガラスのファサードに細長い影を落としている。ワシの下降気流が環境と相互作用し、屋上の埃や紙くずが風で舞い上がる。

ワシは近代的なコンクリートの高層ビルの屋上へと近づく。制御された羽ばたきによって徐々に勢いを失い、バランスを取るために筋肉の緊張と微調整が見て取れるように鉤爪を前方に伸ばす。「Ambition」と書かれた大きな金属製の屋上看板にしっかりと着地する。鉤爪が金属表面を掴み、わずかな振動と金属音が響く。体重が前方に移動し、その後安定する。羽毛は自然に落ち着く。ワシはわずかな頭の微動と瞬きで地平線をスキャンする。

安定した時間的連続性を維持すること。不自然なフレーム補間は避けること。誇張されたスローモーションはなし。現実世界の物理法則、正確な体重移動、本物の鳥の解剖学、力に対する環境の反応を表現すること。
```
