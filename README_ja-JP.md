## 🤔 Seedance 2.0 とは？

**Seedance 2.0** は**ByteDance**が開発した動画生成モデルで、業界初の**4モーダル同時入力**をサポートするモデルです — 画像、動画、音声、テキスト。

**Key Features:**
- 🎥 **テキストから動画** — テキスト説明から動画を生成
- 🖼️ **画像から動画** — 静止画をダイナミックな動画にアニメーション化
- 📹 **動画から動画** — 既存の動画を変換・拡張
- 🎵 **音声駆動** — 音声入力で動画を生成
- 📐 **最大1080p解像度**、4〜15秒の長さ
- 🔊 **自動吹き替え＆スコアリング** — 自動ナレーションとBGM

### No. 3: Demon Slayer ライブアクションバトルプロンプト for Seedance 2.0

![中文](https://img.shields.io/badge/lang-中文-red)
![Featured](https://img.shields.io/badge/⭐-Featured-gold)

#### 📖 説明

Seedance 2.0 用の、詳細で高エネルギーな動画プロンプト。鬼滅の刃風のバトル（水の呼吸 vs 雷の呼吸）を実写で15秒間にわたって描く。プロンプトでは、スタイル（ハリウッド実写漫画化、ダークサムライ、4K、極端なクイックカット、パーティクルライトエフェクト）、設定（夜の霧深い森）、そしてキャラクターのアクション、パワーアップシーケンス、最終的な激突を詳述する3つの異なるショットが指定されている。

#### 📝 プロンプト

```
実写漫画化 · 呼吸法決戦（15秒 · 超絶エフェクト版）
【核心的焦点】：水の呼吸（青い水龍）VS 雷の呼吸（黄金の稲妻）、実写による超高速デュエル。
【スタイル】：ハリウッド実写漫画化映画のクオリティ、ダークな侍スタイル、4K 超高画質、超高速カット、パーティクル光エフェクト爆発、ゴア表現なし。
【尺】：15秒
【シーン】：月明かりの霧深い森、ぬかるんだ地面、落ち葉。
[00:00-00:05] ショット 1：水ノ旋律 · 序ノ型（チャージ感）
ビジュアル：緑と黒の市松模様の羽織をまとった若い侍が、月明かりの下で重心を低くし、両手で刀を構える。
アクション：彼が深く息を吸い込むと、周囲の空気が瞬時に凝固する。刀を抜くと同時に、高圧の水流が凝縮された巨大な青い水龍が空中に現れ、彼の体と刀の周りを高速で回転し、水が流れる轟音を発する。
特殊効果の詳細：水流にはリアルな水しぶきの感覚があり、暗い森を照らす。
[00:05-00:10] ショット 2：雷ノ閃光 · 進撃（超高速感）
ビジュアル：向かい合う敵、黄色の三角模様の羽織をまとった金髪の剣士が、極端に体を低くし、居合術の構えを取る。
アクション：地面が突然爆発する。彼の全身は瞬時にまばゆい黄金の稲妻の残像と化し、肉眼では見えない速度で木々の間を「Z」字型に屈折しながら高速で進撃する。
特殊効果の詳細：彼が通過した場所には、黄金の電弧と焦げた落ち葉が残る。
[00:10-00:15] ショット 3：水雷激突 · 終ノ響（奥義の衝突）
ビジュアル：超高速の正面衝突。若い侍は巨大な青い水龍を振り下ろして攻撃を迎え撃ち、稲妻と化した金髪の剣士がそれに激突する。
アクション：フレーム中央で二本の刀が激しく衝突する。
特殊効果のスペクタクル：青い水龍と黄金の稲妻が瞬時に爆発し、巨大な水雷エネルギーの嵐となって外側へ広がる。周囲の大木はエネルギー波によって真っ二つに折れ、泥、水、光がカメラを覆い隠す。シーンは極めてまばゆい青、黄、白の光の中で終わる。

> 素早い衣装チェンジの12秒動画コレクションを生成するためのプロンプトです。キャラクターの顔と骨格は参照画像と100%一致している必要があります。このシーケンスでは、「ハンドフリックトランジション」を使用して、テンポの速いエネルギッシュな音楽トラックに厳密に合わせて、5つの対照的なスタイル（JK ロリータ、ブリティッシュ、ロリータ、OL、ダーク/サブカルチャー）を切り替えます。

#### 📝 プロンプト

```
キャラクターは参照画像と完全に一致させ、顔の特徴を 100% 同一にし、骨格、目の間隔、顎の形状を維持し、美化や年齢変更は禁止します。12 秒クイックチェンジコレクション：純粋な黒の背景で、5 種類の全く異なる対照的なルック、リラックスした自然な表情、（JK ロリータスタイル、ブリティッシュスタイル、ロリータ、OL、ダークスタイル / ライトクールサブカルチャースタイル）、ビートに合わせて厳密にタイミングを合わせた「ハンドフリックトランジション」を使用します。非常に速いリズム、極端なコントラストで、エンディングは最初のルックに戻ります。
```
