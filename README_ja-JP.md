## 🤔 Seedance 2.0 とは？

**Seedance 2.0** は**ByteDance**が開発した動画生成モデルで、業界初の**4モーダル同時入力**をサポートするモデルです — 画像、動画、音声、テキスト。

**Key Features:**
- 🎥 **テキストから動画** — テキスト説明から動画を生成
- 🖼️ **画像から動画** — 静止画をダイナミックな動画にアニメーション化
- 📹 **動画から動画** — 既存の動画を変換・拡張
- 🎵 **音声駆動** — 音声入力で動画を生成
- 📐 **最大1080p解像度**、4〜15秒の長さ
- 🔊 **自動吹き替え＆スコアリング** — 自動ナレーションとBGM

### ユニバーサル シネマティック ビデオ スクリプト/プロンプト フレームワーク

![中文](https://img.shields.io/badge/lang-中文-red)

> 15秒のプロモーション動画を生成するための汎用的なフレームワーク（スクリプト/プロンプト）で、雰囲気作り（ゆっくり）、緊張感の構築（ゆっくりから速く）、視覚的アンカーのモンタージュ（高速カット）、そしてテキストオーバーレイによるゆっくりとした物語的な結論という4部構成に従います。このスクリプト例は、成都のプロモーション動画用です。

#### 📝 プロンプト

```
シーン 1 0～2秒 エクストリームクローズアップ。
早朝、年配の人物の手が、熱いお茶の入った碗を青石のテーブルの上でゆっくりと押す。湯気が立ち上り、窓の外の薄い朝霧と混じり合う。静寂な情景が、呼吸するようなリズムで流れる。

シーン 2 2～5秒 ローアングルワイドショット。
カメラは濡れた青石の道に近く、一列の人物が路地の奥からゆっくりと歩み出てくる。両側の古い壁は苔で覆われ、赤い提灯が朝の風に優しく揺れる。路地の入り口から光が差し込み、逆光の中、金色の塵が舞う。

シーン 3 5～8秒 ミディアムショット、ゆっくりとプッシュイン。

シーン 4 8～11秒 高速モンタージュ、1フレームあたり約0.5秒

エンドクレジット：成都・悠久の豊穣の地。黒画面。終了。
