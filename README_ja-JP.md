## 🤔 Seedance 2.0 とは？

**Seedance 2.0** は**ByteDance**が開発した動画生成モデルで、業界初の**4モーダル同時入力**をサポートするモデルです — 画像、動画、音声、テキスト。

**Key Features:**
- 🎥 **テキストから動画** — テキスト説明から動画を生成
- 🖼️ **画像から動画** — 静止画をダイナミックな動画にアニメーション化
- 📹 **動画から動画** — 既存の動画を変換・拡張
- 🎵 **音声駆動** — 音声入力で動画を生成
- 📐 **最大1080p解像度**、4〜15秒の長さ
- 🔊 **自動吹き替え＆スコアリング** — 自動ナレーションとBGM

![中文](https://img.shields.io/badge/lang-中文-red)
![Featured](https://img.shields.io/badge/⭐-Featured-gold)

#### 📖 説明

Seedance 2.0 用の、詳細で高エネルギーな動画プロンプト。鬼滅の刃風のバトル（水の呼吸 vs 雷の呼吸）を実写で15秒間にわたって描く。プロンプトでは、スタイル（ハリウッド実写漫画化、ダークサムライ、4K、極端なクイックカット、パーティクルライトエフェクト）、設定（夜の霧深い森）、そしてキャラクターのアクション、パワーアップシーケンス、最終的な激突を詳述する3つの異なるショットが指定されている。

#### 📝 プロンプト

```
実写漫画化 · 呼吸法決戦（15秒 · 超絶エフェクト版）
【核心的焦点】：水の呼吸（青い水龍）VS 雷の呼吸（黄金の稲妻）、実写による超高速デュエル。
【スタイル】：ハリウッド実写漫画化映画のクオリティ、ダークな侍スタイル、4K 超高画質、超高速カット、パーティクル光エフェクト爆発、ゴア表現なし。
【尺】：15秒

ビジュアル：緑と黒の市松模様の羽織をまとった若い侍が、月明かりの下で重心を低くし、両手で刀を構える。
アクション：彼が深く息を吸い込むと、周囲の空気が瞬時に凝固する。刀を抜くと同時に、高圧の水流が凝縮された巨大な青い水龍が空中に現れ、彼の体と刀の周りを高速で回転し、水が流れる轟音を発する。

[00:05-00:10] ショット 2：雷ノ閃光 · 進撃（超高速感）

特殊効果のスペクタクル：青い水龍と黄金の稲妻が瞬時に爆発し、巨大な水雷エネルギーの嵐となって外側へ広がる。周囲の大木はエネルギー波によって真っ二つに折れ、泥、水、光がカメラを覆い隠す。シーンは極めてまばゆい青、黄、白の光の中で終わる。
```

> Seedance 2.0 向けの、女性と高級車をフィーチャーしたハイエンドなコマーシャルビデオを生成するために設計された、詳細なマルチシーンプロンプト。ウィップパン、ラピッドズームイン、手持ちカメラの揺れといった高度な映画撮影技術を駆使し、ダイナミックなトランジションと視覚的なインパクトを生み出します。このプロンプトは、シーンの長さ、カメラの動き、キャラクターの外見（参照画像と一致）、そして全体的な美学を具体的に指定しています。

#### 📝 プロンプト

```
件名: @画像1 の女性、高い顔の特徴の一貫性を維持し、顔の歪みがなく安定しており、自信に満ちた鋭い目つきで。
1. (0-2秒) クローズアップ: ドレッサーのジュエリーを細い指がなぞる。すぐにビンテージの赤い口紅を塗るシーンに切り替わり、鮮明な動作。
2. (2-4秒) クローズアップ: イブニングドレスのジッパーが素早く引き上げられるダイナミックなショット。黒いスティレットヒールを履く横顔に切り替わる。
3. (4-6秒) シーン定義: 女性は [ハイエンドの黒いシルクのストラップレスのトレーリングガウンを着用し、スカート全体に手刺繍の豪華な金糸の刺繍が施され、スワロフスキーのラインストーンが密に散りばめられ、豪華で流れるような質感]、ハンドバッグを手にドアから出ていく。
4. (6-9秒) シーン転換: 赤いスーパーカーのエンジンが始動し、ダッシュボードの針が跳ね上がる。すぐに夜の都市トンネルを高速で走る車の車輪に切り替わり、映像にはわずかな手ブレがある。
5. (9-12秒) シーン転換: 車のドアが開き、長く細い脚がレッドカーペットに降り立つ。カメラが素早く引いて (Rapid Pull Out)、星がちりばめられた会場を映し出す。
6. (12-15秒) クライマックス: 密集したメディアのフラッシュに直面し、女性は優雅に振り返り、スカートは水のように流れ、スーパースターの笑顔を見せる。: Whip Pan を使用してシーンを接続し、Rapid Zoom In と組み合わせて細部を強調する。映像はゴーストがなく安定している。: 4K Ultra HD、シネマティックな黒と金のカラーグレーディング、シャープな透明度、フィルムグレインテクスチャ。: 通常の人間の解剖学、@画像1 の顔は歪んでおらず、画面上のアクションはオーディオの速いリズムと完全に一致している。
```
