## 🤔 什麼是 Seedance 2.0？

**Seedance 2.0** 是**字節跳動**開發的視頻生成模型，是業內首個支持**四模態同時輸入**的模型——圖片、視頻、音頻和文本。

**Key Features:**
- 🎥 **文本生成視頻** — 根據文字描述生成視頻
- 🖼️ **圖片生成視頻** — 將靜態圖片動畫化為動態視頻
- 📹 **視頻轉視頻** — 轉換和擴展現有視頻
- 🎵 **音頻驅動** — 通過音頻輸入驅動視頻生成
- 📐 **最高 1080p 分辨率**，4-15 秒時長
- 🔊 **自動配音配樂** — 自動旁白和背景音樂

### 電影打鬥場景提示：白西裝男 vs. 西伯利亞長毛貓

```

第二鏡：切換至無人機高空廣角鏡頭。

就在白衣男子從容舉杯，西伯利亞長毛貓溫順地喵叫的瞬間，

白衣男子手腕一翻，酒杯穩穩落地，沒有破碎，身體瞬間下沉，擺出戰鬥姿態。
先前的優雅與從容蕩然無存，取而代之的是經驗豐富的戰士緊繃的肌肉和殺意。
石窟內的溫度驟降，薄霧被兩股對峙的氣場攪動翻騰。
前一秒，那是無害的目光；下一秒，已成為一場生死對決。
空氣凝固，只聽見急促的呼吸聲和細小石子滾落的清脆聲。

毛髮豎起：細微的「唰」毛髮震動聲
利爪彈出：尖銳的「咔」聲
西裝男肌肉繃緊：布料收緊的「吱」聲
酒杯輕輕放在桌上：清脆的「叮」聲
石窟氣場壓力：低沉、深邃的空氣震動「嗡——」
```
