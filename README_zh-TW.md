## 🤔 什麼是 Seedance 2.0？

**Seedance 2.0** 是**字節跳動**開發的視頻生成模型，是業內首個支持**四模態同時輸入**的模型——圖片、視頻、音頻和文本。

- 📹 **視頻轉視頻** — 轉換和擴展現有視頻
- 🎵 **音頻驅動** — 通過音頻輸入驅動視頻生成
- 📐 **最高 1080p 分辨率**，4-15 秒時長
- 🔊 **自動配音配樂** — 自動旁白和背景音樂

### Seedance 2.0 提示：「斬仙」標題動畫

![中文](https://img.shields.io/badge/lang-中文-red)

> 為 Seedance 2.0 製作一個高度詳細、帶有時間碼的提示，以生成一個動態、電影感的標題動畫，內容為「斬仙」。此動畫將融合水墨、冰和玉石的紋理，強調視覺張力及強烈的青藍色調。使用者也建議將此邏輯應用於其他風格，例如變形金剛風格的標題。

#### 📝 提示詞

