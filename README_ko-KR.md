## 🤔 Seedance 2.0이란?

**Seedance 2.0**은 **ByteDance**가 개발한 비디오 생성 모델로, 업계 최초로 **4가지 모달 동시 입력**을 지원합니다 — 이미지, 비디오, 오디오, 텍스트.

**Key Features:**
- 🎥 **텍스트에서 비디오** — 텍스트 설명으로 비디오 생성
- 🖼️ **이미지에서 비디오** — 정적 이미지를 동적 비디오로 애니메이션화
- 📹 **비디오에서 비디오** — 기존 비디오 변환 및 확장
- 🎵 **오디오 기반** — 오디오 입력으로 비디오 생성
- 📐 **최대 1080p 해상도**, 4-15초 길이
- 🔊 **자동 더빙 및 스코어링** — 자동 나레이션 및 배경 음악

### 사바나 하드팩 위의 미래형 호버크래프트

![English](https://img.shields.io/badge/lang-English-blue)

> 미래형 호버크래프트가 사바나를 가로질러 질주하는 10초 길이의 연속 촬영 비디오를 생성하기 위한 매우 상세하고 긴 프롬프트입니다. 여기에는 특정 카메라 움직임(공중 궤도, 급강하, 탑다운 잠금, 측면 추격, 근접 나선)과 환경 세부 사항(더스트 데블, 폭풍선, 흰개미집)이 포함됩니다. 이 프롬프트는 Seedance 2.0이 아닌 Kling 2.6에서 사용되었지만, 고급 비디오 생성 프롬프트의 맥락으로 포함되었습니다.

#### 📝 프롬프트

```
사바나의 단단한 지면 위를 먼지 회오리와 함께 질주하는 미래형 호버크라프트 (10초, 단일 연속 촬영)
```
