## 🤔 Seedance 2.0이란?

**Seedance 2.0**은 **ByteDance**가 개발한 비디오 생성 모델로, 업계 최초로 **4가지 모달 동시 입력**을 지원합니다 — 이미지, 비디오, 오디오, 텍스트.

**Key Features:**
- 🎥 **텍스트에서 비디오** — 텍스트 설명으로 비디오 생성
- 🖼️ **이미지에서 비디오** — 정적 이미지를 동적 비디오로 애니메이션화
- 📹 **비디오에서 비디오** — 기존 비디오 변환 및 확장
- 🎵 **오디오 기반** — 오디오 입력으로 비디오 생성
- 📐 **최대 1080p 해상도**, 4-15초 길이
- 🔊 **자동 더빙 및 스코어링** — 자동 나레이션 및 배경 음악

검객은 경공술로 대나무 끝을 밟고, 대나무는 그녀의 발아래서 휘어지고 흔들린다. 그녀는 여러 대나무 줄기 사이를 계속해서 뛰어다니고, 그녀의 붉은 옷은 푸른 대나무 바다에서 특히 눈에 띈다. 그녀가 대나무 줄기를 밟을 때마다 대나무는 휘어지고 떨린다.
맹인 검객은 대나무 끝 사이를 쫓으며 바싹 뒤따른다. 그는 긴 봉을 휘둘러 검객이 밟았던 대나무 줄기를 치고 부러뜨리며, 대나무는 "파" 하고 부러져 날아간다.

1-2초: 낮은 앵글 샷, 대나무 끝 사이를 뛰어다니는 검객, 대나무가 휘어짐
